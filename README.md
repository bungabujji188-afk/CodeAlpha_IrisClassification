# Iris Flower Classification 🌸

## Description
This project is part of my Data Science Internship.

It classifies iris flowers into:
- Setosa
- Versicolor
- Virginica

## Tools Used
- Python
- Pandas
- Scikit-learn
- Seaborn

## Model Used
Random Forest Classifier

## Accuracy
96.6%

## Conclusion
The model successfully predicts flower types with high accuracy.
